# HotelManagementSystem 测试用例报告

> 项目：酒店信息管理系统
> 技术栈：Java 17 + Tomcat 10.1 + MySQL 8.0 + HikariCP
> 生成日期：2026-05-19

---

## 一、测试范围

| 模块 | 涉及文件 | 测试重点 |
|------|----------|----------|
| 用户认证 | `LoginServlet`, `RegisterServlet`, `AuthFilter`, `UserDAO` | 登录验证、角色校验、注册流程、会话管理 |
| 房间管理 | `RoomDAO`, `RoomStatusServlet`, `AllRoomStatusServlet`, `UpdateRoomStatusServlet` | 状态查询、状态更新、空闲/入住房间筛选 |
| 入住退房 | `CheckInServlet`, `CheckInFormServlet`, `CheckOutServlet`, `BookingDAO` | 事务一致性、费用计算、权限验证 |
| 首页统计 | `IndexServlet`, `RoomDAO`, `BookingDAO` | 数据统计准确性 |
| 数据库连接 | `DatabaseConnection`, `DBConfig` | 连接池初始化、连接泄漏 |

---

## 二、测试环境

| 项 | 配置 |
|----|------|
| JDK | 17+ |
| Tomcat | 10.1.x |
| MySQL | 8.0.x |
| 浏览器 | Chrome / Edge（最新版） |
| 数据库初始状态 | 执行 `schema.sql` 后的标准状态 |
